# Getting Started with Create React App

### Run in dev mode :

- npm i
- npm start

###  Run in build mode : 

- npm i
- npm install -g serve
- npm run build
- serve -s build

###  Run in Test mode : 

- npm i -D jest
- npm i -D @testing-library/react
- npm i -D @types/jest
- npm i -D babel-jest @babel/preset-env @babel/preset-react
- npm i -D @testing-library/dom
- npm i -D @testing-library/jest-dom
- npx jest
